צור פרויקט Firebase חינמי
1. כנס ל-console.firebase.google.com
2. לחץ Add project ← תן שם לפרויקט (למשל: orot-library)
3. לחץ Continue ← Continue ← Create project
🗄️
הפעל מסד נתונים
בתפריט השמאלי של Firebase:
1. לחץ Build (או All products)
2. בחר Realtime Database
3. לחץ Create database
4. בחר מיקום (ברירת מחדל בסדר) ← לחץ Next
5. בחר "Start in test mode" ← לחץ Enable
📋
הדבק את קוד ה-Config
1. בFirebase: לחץ ⚙️ Project settings
2. גלול מטה ← לחץ Add app ← בחר </> (Web)
3. תן שם ← לחץ Register app
4. סמן הכל ← העתק ← הדבק כאן 👇
